# File Filter Tool

This is a simple Python application with a graphical user interface (GUI) for filtering text files. It allows you to quickly find specific lines using a keyword or regular expression (Regex) and save the results into a separate file.

## How to Install and Run

1. Ensure you have Python installed on your computer.
2. Download all project files into a single folder.
3. Open a terminal in this folder and install the dependencies (needed for testing):
   ```bash
   pip install -r requirements.txt
